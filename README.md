# Campus Link

대명캠의 아트/영상/애니메이션 인력과 성서캠의 개발/기획 인력을 연결하는 학교 기반 협업 매칭 웹서비스입니다.

## 핵심 목표

Campus Link는 에브리타임 게시글만으로는 확인하기 어려운 작업물, 역할, 팀 준비 상태, 협업 가능 여부를 구조화해 캠퍼스 간 협업을 쉽게 만드는 것을 목표로 합니다.

## 3주 MVP 범위

- 회원가입/로그인
- 학교 이메일 인증 UI
- 온보딩 5단계
- 포트폴리오 등록
- 프로젝트 등록
- 프로젝트 목록/상세
- 검색/필터
- 지원하기/제안하기
- 지원 현황 확인

## 기술 스택

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Spring Boot, Java
- Database: Supabase PostgreSQL
- Storage: Supabase Storage
- Deploy: Vercel, Render/Railway 또는 로컬 시연

## 팀 역할

- PM/기획: 기능 정의, 화면 흐름, API/DB 명세, 일정 관리
- Frontend: Next.js 화면 구현 및 API 연결
- Backend: Spring Boot API 및 DB 연동
- Test/Data: 더미데이터, 테스트 체크리스트, 발표 자료 보조
