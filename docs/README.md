# Docs Folder

This folder contains planning, architecture, and AI collaboration rules.

## Document Map

| File | Purpose |
| --- | --- |
| `campus-link-spec.md` | Product planning spec and MVP scope |
| `campus-link-ui.html` | Static HTML UI prototype |
| `architecture-ddd.md` | DDD boundaries and code ownership rules |
| `ai-task-template.md` | Template for assigning one feature to an AI agent |
| `feature-slices.md` | 3-week MVP broken into small feature slices |
| `development-setup.md` | Local environment setup guide |

## How to Use This Folder

- PM starts with `campus-link-spec.md`.
- Developers start with `development-setup.md`.
- AI-assisted development starts with `architecture-ddd.md` and `ai-task-template.md`.
- Sprint planning starts with `feature-slices.md`.
